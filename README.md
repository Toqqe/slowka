## Polskie "Wordle"

Utworzone za pomocą Reacta + ReactBootstrap

[Demo](https://slowka.szczapam.pl)


W folderze projektu, można lokalnie wystartować grę:

```
npm run dev
```
Po uruchomieniu będzie dostępna pod adresem:
http://localhost:5173/

Zbudowanie aplikacji na produkcję:
```
npm run build
```
